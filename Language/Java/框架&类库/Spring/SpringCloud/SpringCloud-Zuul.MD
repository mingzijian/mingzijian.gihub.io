Spring Cloud Zuul
---------------------------
路由网关。

Zuul是一个L7应用程序网关，提供动态路由、监视、恢复、安全等功能。
Zuul是Netflix的基于JVM的路由器和服务器端负载均衡器。

### 替代

建议使用`Spring Cloud Gateway`来替代`Spring Cloud Zuul`

[spring-cloud-gateway](https://spring.io/projects/spring-cloud-gateway)
