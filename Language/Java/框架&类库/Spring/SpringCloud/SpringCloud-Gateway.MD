Spring Cloud Gateway
----------------------------------
该项目提供了一个用于在Spring WebFlux之上构建API网关的库。
Spring Cloud Gateway旨在提供一种简单而有效的方法来路由到API，并为它们提供跨领域关注，例如：安全性，监视/指标和弹性。


### 特性
- 基于Spring Framework 5 , Spring Boot 2.0
- 能够匹配任何请求属性上的路由 
- 特定于路由的断言和过滤器
- 断路器集成
- Spring Cloud DiscoveryClient集成
- 易于编写断言和过滤器
- 请求速率限制
- 路径重写
